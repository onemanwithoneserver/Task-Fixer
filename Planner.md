# Daily Planner Schedule

This document outlines the daily schedules configured in the Task Fixer application.

## Schedule Types

The planner supports 5 different schedule types:

1. **Weekday** - Office Flow (Monday, Wednesday, Thursday, Friday)
2. **Tuesday** - Updated Detailed Schedule
3. **Saturday** - Sprint Mode
4. **Holiday Tuesday** - Rest & Reflection (Sunday)
5. **Holiday** - Free Flow (Holidays)

## Categories Overview

The planner organizes tasks into the following categories:

- **Morning Routine** - Wake up activities
- **Health** - Exercise and physical activities
- **Organization** - Task batching and planning
- **Spiritual** - Prayer and meditation
- **Learning** - English and general learning
- **Technical Learning** - MERN Stack, Accessibility Testing
- **Design Learning** - UI/UX Design study
- **Design Practice** - UI/UX Design application
- **Technical Review** - Concept recap and review
- **Travel** - Commute and transit time
- **Meal** - Breakfast, lunch, dinner
- **Work** - Office hours
- **Rest** - Breaks and downtime
- **Planning** - Finance and planning activities
- **Sleep** - Sleep preparation
- **Night Routine** - End of day activities

## Features

### Automatic Day Type Detection
The planner automatically detects the current day and loads the appropriate schedule:
- Sunday â†’ Holiday Tuesday
- Tuesday â†’ Tuesday Schedule
- Saturday â†’ Saturday Schedule
- Other weekdays â†’ Weekday Schedule

### Task Tracking
- Each task can be marked as complete with a checkbox
- Completed tasks show with strikethrough and reduced opacity
- Progress is tracked and synced with the global app state

### Deep Work Highlighting
Tasks containing "DSA" or "MERN" are highlighted as "Deep Work" sessions with:
- Special visual styling (emerald gradient background)
- Animated pulse indicator
- Enhanced border effects

### Time Display
- Real-time clock showing current date and time
- Updates every minute
- 12-hour format with AM/PM

### Schedule Switching
Users can manually switch between different schedule types using the dropdown selector, with visual indicators showing:
- Current active schedule
- "Today" badge for the schedule matching the current day
- Smooth animations for dropdown interactions

## Component Structure

### Props
- `state` - Global state object containing planner data
- `setState` - Function to update global state

### State Management
- `isOpen` - Controls dropdown visibility
- `currentDateTime` - Tracks current time (updates every minute)
- Tasks persist in global state as `state.planner` array
- Day type persists in global state as `state.dayType`

### Data Structure
Each task object contains:
```javascript
{
  time: "04:30 AM â€“ 04:45 AM",
  task: "Wake up",
  category: "Morning Routine",
  done: false  
}
```

## Usage Notes

1. **Schedule Persistence**: Selected schedule and task completion states are stored in the global state
2. **Auto-Reset**: When changing schedule types, all tasks reset to incomplete
3. **Responsive Design**: Fully responsive layout optimized for mobile and desktop
4. **Animations**: Smooth transitions powered by Framer Motion
5. **Accessibility**: Keyboard navigation and screen reader support included
